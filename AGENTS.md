# Табуретка — Мебельный сайт

## Проект
Сайт мебельного магазина «Табуретка». Каталог, отзывы, примеры работ, о нас, контакты, карта.

## Дизайн-референсы
- **UX**: https://nonton.ru/ — навигация по категориям, карточки товаров, структура шапки
- **VK-группа**: https://vk.com/club177780031 — стиль, палитра, настроение

## Логотип
- Квадратное полотно с горчично-жёлтым/охристым фоном
- В центре — белый круг
- Внутри круга — минималистичная иконка высокого табурета (барный стул): сиденье в виде горизонтального овала, ножки расходятся книзу, соединены перекладинами; тонкие тёмно-коричневые линии
- Под иконкой: «ТАБУРЕТКА» — крупный шрифт без засечек, заглавные
- Ниже: «магазин мебели» — мелкий шрифт без засечек, строчные
- Стиль: минимализм, скандинавский дизайн

## Цветовая палитра
- Основной тёмный: `#3E2610` (тёмно-коричневый/графитовый)
- Коричневый средний: `#5B3A1A`
- Акцент/бренд: `#C4A24E` (горчично-жёлтый/охристый — цвет фона логотипа)
- Акцент светлый: `#E0C872`
- Фон: `#FDF8F3` (бежевый)
- Белый: `#FFFFFF` (круг логотипа)
- Текст: `#2D2418`
- Текст лёгкий: `#6B5D4F`
- Текст приглушённый: `#9E9184`
- Бордер: `#E8DFD4`

## Архитектура
- Чистый HTML/CSS/JS (без фреймворка)
- Каталог будет интегрирован с 1С (пока пустые макеты)
- Возможна интеграция с SQL (пока не делать)

## Структура страниц (план)
1. **Главная** — hero, категории, популярные товары, преимущества (готово)
2. **Каталог** — полный каталог товаров (будет 1С)
3. **Отзывы**
4. **Примеры работ**
5. **О нас**
6. **Контакты** — с картой

## Категории каталога
Кухни, Столы, Стулья, Шкафы, Диваны, Кровати, Аксессуары

## Файловая структура
```
taburetka/
  index.html    — главная страница
  css/style.css  — стили
  js/main.js     — интерактивность
```
